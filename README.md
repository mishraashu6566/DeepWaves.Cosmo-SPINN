We present a physics-informed U-net based generative framework, which we call **Cosmo-SPINN**, for wave dark matter simulations governed by Schrödinger–Poisson equations.

We address two complementary tasks:
1. Evolution of cosmological fields from initial conditions to an arbitrary scale factor.
2. Super-resolution of FDM simulations at the specified scale factor.
